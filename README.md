# Cyril Yamoah - Portfolio

My personal portfolio website showcasing my work as a frontend developer. Built with Next.js 15, TypeScript, and Tailwind CSS.

## Features

- Responsive design optimized for all devices
- Blog with MDX support for technical articles
- SEO optimized with structured data (Schema.org)
- Smooth animations and transitions
- Dark mode support
- Accessibility compliant

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Content**: MDX with syntax highlighting (Shiki)
- **Deployment**: [Your deployment platform]

- BPO TECH - Real estate CRM solution

## Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.
